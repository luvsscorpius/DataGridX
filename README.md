# DataGridX - Gerenciamento de Dados de Pessoas

## Descrição

DataGridX - Gerenciamento de Dados de Pessoas é uma aplicação web moderna construída em JavaScript que oferece uma interface de usuário altamente interativa e flexível para exibir, editar e gerenciar informações relacionadas a pessoas. Esta aplicação utiliza a tecnologia Fetch para se comunicar com um servidor Node-RED que atua como o backend para operações CRUD (Criar, Ler, Atualizar, Excluir) em dados de pessoas.

## Visão Geral

DataGridX - Gerenciamento de Dados de Pessoas é uma solução eficaz para empresas, organizações e indivíduos que precisam de uma maneira rápida e intuitiva de gerenciar informações sobre pessoas, como clientes, funcionários, membros da equipe, contatos de negócios, etc.

## Principais Recursos

1. DataGrid De Pessoas Interativo: Uma poderosa grade de dados interativa que permite exibir e gerenciar informações detalhadas sobre pessoas, incluindo nome, sobrenome, cidade, número de telefone, e-mail, etc.
2. Integração com Node-RED: O Node-RED atua como o servidor backend para todas as operações de CRUD relacionadas a dados de pessoas. Ele fornece uma API RESTful segura para manipular esses dados no banco de dados subjacente.
3. Operações CRUD Simples: Os usuários podem criar, ler, atualizar e excluir registros de pesssoas diretamente na grade de dados, com todas as atualizações refletidas instantaneamentes no servidor e vice-versa.
4. Suporte a Dados em Tempo Real: A aplicação é capaz de atualizar automaticamente os dados em tempo real à medida que mudanças ocorrem no servidor, garantindo que as informações estejam sempre atualizadas.

## Tecnologias Utilizadas

- Frontend: JavaScript, HTML, CSS, Fetch API, Biblioteca ag-Grid.
- Backend: Node-RED, Node.js e MariaDB.

## Benefícios
- Acesso fácil e rápido aos seus dados em um formato de grade amigável.
- Integração simplificada com sistemas existentes usando 
Node-RED como backend.

DataGridX é a solução definitiva para facilitar o manuseio de grandes conjuntos de dados tabulares em tempo real, e sua integração com o Node-RED torna a implementação e a personalização uma tarefa tranquila. Simplifique suas operações de gerenciamento de dados com DataGridX!
